# 7. Deployment View

### 7.1 Target Environment
- Android mobile device/emulator (API 24+)
- Local Room database inside app sandbox
- Local app-managed photo storage/cache

### 7.2 Build and Delivery
- Gradle-based builds (`assembleDebug`, `bundleRelease` for release paths).
- Planned CI/CD via GitHub Actions.
